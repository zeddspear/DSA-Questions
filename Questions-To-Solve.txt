Interview Coding Questions List
=================================

1. Arrays
----------
- Two Sum
- Best Time to Buy and Sell Stock
- Maximum Subarray (Kadane's Algorithm)
- Product of Array Except Self
- Find the Duplicate Number
- Container With Most Water
- Move Zeroes
- Trapping Rain Water
- Merge Intervals
- Subarray Sum Equals K

2. Strings
----------
- Valid Anagram
- Group Anagrams
- Longest Substring Without Repeating Characters
- Minimum Window Substring
- String to Integer (atoi)
- Longest Palindromic Substring
- Palindrome Permutation
- Regular Expression Matching
- Word Ladder
- Edit Distance

3. Linked Lists
---------------
- Reverse Linked List (Iterative & Recursive)
- Detect Cycle in a Linked List (Floyd’s Tortoise and Hare)
- Merge Two Sorted Lists
- Remove Nth Node From End
- Find the Middle of a Linked List
- Intersection of Two Linked Lists
- Add Two Numbers
- Reorder List

4. Stacks & Queues
------------------
- Valid Parentheses
- Evaluate Reverse Polish Notation
- Min Stack
- Implement Queue using Stacks
- Daily Temperatures
- Largest Rectangle in Histogram
- Sliding Window Maximum

5. Hash Tables
--------------
- Two Sum (Hash Table approach)
- Group Anagrams (Hash Table approach)
- Intersection of Two Arrays / Intersection II
- Find All Duplicates in an Array
- First Unique Character in a String
- Longest Consecutive Sequence

6. Trees & Binary Trees
-----------------------
- Binary Tree Inorder Traversal (Recursive & Iterative)
- Binary Tree Level Order Traversal (BFS)
- Invert/Reverse Binary Tree
- Diameter of Binary Tree
- Lowest Common Ancestor (LCA) for Binary Tree and BST
- Validate Binary Search Tree
- Serialize and Deserialize Binary Tree
- Construct Binary Tree from Preorder and Inorder Traversal
- Path Sum / Sum Root to Leaf Numbers

7. Heaps (Priority Queues)
--------------------------
- Kth Largest Element in an Array
- Merge k Sorted Lists
- Find Median from Data Stream
- Top K Frequent Elements
- Kth Smallest Element in a Sorted Matrix
- Rearrange String k Distance Apart

8. Graphs
---------
- Depth First Search (DFS) & Breadth First Search (BFS)
- Number of Islands
- Clone Graph
- Course Schedule (Cycle Detection / Topological Sort)
- Topological Sort
- Word Ladder
- Shortest Path in Unweighted Graph (BFS)
- Dijkstra’s Algorithm (Weighted Graphs)
- Detect a Cycle in a Directed Graph (DFS/BFS)
- Minimum Spanning Tree (Prim’s and Kruskal’s)
- Connected Components in a Graph

9. Dynamic Programming (DP)
---------------------------
- Fibonacci Sequence (Iterative / Recursive / DP)
- Climbing Stairs
- Longest Increasing Subsequence (LIS)
- 0/1 Knapsack Problem
- Longest Common Subsequence (LCS)
- Coin Change Problem
- House Robber (and House Robber II)
- Word Break (I & II)
- Partition Equal Subset Sum
- Unique Paths (and Unique Paths II with obstacles)
- Edit Distance (Levenshtein Distance)
- Minimum Path Sum (Grid based)

10. Backtracking
----------------
- Permutations
- Combinations (n choose k)
- Subsets (Power Set)
- N-Queens Problem
- Sudoku Solver
- Word Search
- Combination Sum (I & II)
- Generate Parentheses

11. Sorting & Searching
------------------------
- Merge Sort
- Quick Sort
- Binary Search & its Variants (e.g., Search in Rotated Sorted Array, Find Peak Element)
- Quickselect (Kth Largest/Smallest Element)
- Sort Colors (Dutch National Flag Problem)
- Median of Two Sorted Arrays
- Top K Frequent Elements (Alternate approaches: bucket sort, sorting)

12. Bit Manipulation
--------------------
- Single Number (I & II variants)
- Hamming Distance
- Counting Set Bits (Brian Kernighan’s Algorithm)
- Power of Two
- Reverse Bits
- Missing Number
- Find the Two Non-Repeated Numbers in an Array
- Number of 1 Bits (Population Count)
- Bitwise AND of Numbers Range

13. Mathematics & Number Theory
-------------------------------
- Sieve of Eratosthenes (Prime Numbers)
- Greatest Common Divisor (GCD using Euclid’s Algorithm)
- Count Primes
- Factorial Trailing Zeroes
- Permutations & Combinations
- Fibonacci & Recurrence Relations
- FizzBuzz

14. Greedy Algorithms
---------------------
- Activity Selection / Interval Scheduling
- Job Sequencing with Deadlines
- Fractional Knapsack Problem
- Huffman Coding
- Gas Station Problem
- Merge Intervals (greedy solution)
- Meeting Rooms / Minimum Meeting Rooms
